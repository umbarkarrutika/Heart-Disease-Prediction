Input Field Information
  • Age
    📥 Enter age in years (e.g., 45)
    📏 Ask the patient or check ID records.
  • Sex
    📥 1 = Male, 0 = Female
    📏 Based on biological sex.
  • Chest Pain Type
    📥 0–3
    📏 Ask patient:
    • 0: Typical (with exertion)
    • 1: Atypical (not exertion-related)
    • 2: Non-anginal
    • 3: No pain
  • Resting BP (trestbps)
    📥 Value in mm Hg (e.g., 130)
    📏 Use a BP machine when relaxed.
  • Cholesterol
    📥 mg/dL (e.g., 210)
    📏 Blood test at lab/clinic.
  • Fasting Blood Sugar (fbs)
    📥 1 if >120 mg/dL, else 0
    📏 Blood test after 8-hr fast.
  • ECG (restecg)
    📥 0–2
    📏 From ECG report:
    • 0: Normal
    • 1: ST-T abnormal
    • 2: Left ventricular hypertrophy
  • Max Heart Rate (thalach)
    📥 Max BPM during test (e.g., 160)
    📏 Measured during treadmill or stress test.
  • Exercise-Induced Angina (exang)
    📥 1 = Yes, 0 = No
    📏 Ask patient during physical test.
  • Oldpeak
    📥 Decimal (e.g., 2.3)
    📏 From stress ECG report.
  • Slope
    📥 0 = Upsloping, 1 = Flat, 2 = Downsloping
    📏 From ECG/stress test analysis.
  • Cardiac Arrest (ca)
    📥 1 = Yes, 0 = No
    📏 Based on doctor’s diagnosis/patient history.
  • Thalassemia (thal)
    📥 1 = Normal, 2 = Fixed, 3 = Reversible
    📏 From thalassemia blood test or medical file.
